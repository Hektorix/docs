## What is Hektorix?

Hektorix is a cybersecurity monitoring platform designed specifically for SMBs who need actionable security insights without requiring deep technical expertise. Our solution combines automated vulnerability detection with clear, prioritized reporting to help you protect your business.

## How It Works

<Steps>
  <Step title="Install the Agent">
    Deploy our lightweight Rust agent on your Windows workstations in just a few clicks
  </Step>
  <Step title="Automatic Data Collection">
    The agent securely sends system information and software inventory to our servers daily
  </Step>
  <Step title="Vulnerability Analysis">
    Our backend processes your data against the latest CVE database to identify security risks
  </Step>
  <Step title="Clear Reporting">
    View prioritized vulnerabilities and system insights through our intuitive web dashboard
  </Step>
</Steps>

## Key Features

### System Monitoring

- **Disk & Memory Usage**: Track storage and memory consumption across your fleet
- **System Information**: Complete inventory of OS versions, hardware specs, and configurations
- **Windows Registry**: Monitor critical system settings and configurations

### Security Management

- **Software Inventory**: Comprehensive list of installed applications with version tracking
- **Vulnerability Detection**: Automated scanning against the National Vulnerability Database (NVD)
- **Risk Prioritization**: Focus on the vulnerabilities that matter most to your environment

### Simple Deployment

- **Easy Installation**: One-click installer with minimal system requirements
- **Automatic Updates**: Self-updating agent ensures you always have the latest features
- **Silent Deployment**: Support for mass installation across multiple workstations
